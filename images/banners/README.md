# バナー広告画像フォルダ

このフォルダにバナー広告画像を保存してください。

## 推奨ファイル名
- `banner-01.jpg` - メインバナー（横長、目立つ位置）
- `banner-02.jpg` - サブバナー1
- `banner-03.jpg` - サブバナー2

## 推奨画像サイズ
- **メインバナー**: 1200×400px (3:1比率)
- **サブバナー**: 600×400px (3:2比率)

## ファイル形式
- `.jpg` - 写真系画像
- `.png` - ロゴやテキスト入り画像
- `.webp` - 最適化重視

## 注意事項
- ファイルサイズは500KB以下推奨
- 高解像度（Retina対応）を考慮した画像を使用
- alt属性用の適切な説明テキストを準備
