# 代表挨拶ページ用写真

## 使用される画像
`images/representative-photo.jpg`

## 推奨画像サイズ
- 横幅: 1200px～1600px
- 縦幅: 800px～1000px
- アスペクト比: 3:2 または 16:10
- ファイル形式: JPG
- ファイルサイズ: 1MB以下（圧縮推奨）

## 画像の種類
- 代表者の写真
- 会社の集合写真
- 施工現場の写真
- オフィスの写真
など、代表挨拶に関連する写真

## 表示位置
挨拶文の下、署名の前に表示されます。

## 画像の変更方法
1. `images/` フォルダに `representative-photo.jpg` という名前で画像を保存
2. ブラウザで代表挨拶ページを再読み込み（Ctrl+Shift+R または Cmd+Shift+R）
